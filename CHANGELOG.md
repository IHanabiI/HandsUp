# HandsUp v1.1.0 更新日志 / Changelog

发布日期：2026-06-22

## 中文

- 修复故障机器人导致的牌组/牌库异常问题。
- 适配 Slay the Spire 2 v0.107.1 与 BaseLib v3.3.2 的当前加载接口。
- 修复 Steam 创意工坊版本中 DLL 初始化失败的问题。
- 在主页模组设置中加入 HandsUp 快捷键设置入口。
- 修复模组设置与操作设置中的快捷键显示不同步问题。
- 优化举手手菜单显示：单机隐藏底部状态日志，联机保留申请/等待等状态提示。
- 将“从头再来”移动到按钮列表最后，并用更大间距降低误触风险。
- 建议所有通过创意工坊或手动安装的玩家更新到 v1.1.0。

## English

- Fixed a deck/library issue caused by the faulty robot interaction.
- Updated compatibility for Slay the Spire 2 v0.107.1 and BaseLib v3.3.2.
- Fixed a DLL initialization failure in the Steam Workshop build.
- Added the HandsUp hotkey settings entry to the main-menu mod configuration screen.
- Fixed hotkey display synchronization between mod configuration and controls settings.
- Improved the HandsUp menu display: singleplayer now hides the bottom status log, while multiplayer keeps approval/waiting status feedback visible.
- Moved "Restart Run" to the bottom of the action list with extra spacing to reduce accidental clicks.
- Players using either Steam Workshop or manual installation are recommended to update to v1.1.0.
